Extensions and Utilities

Below is a list of free code extensions and utilities that you may find useful.

CpReflection - useful extensions for C# reflection functions. If you need to do a lot of that - check this out!

CpCodeExtensions - C# code extensions. Includes UidGenerator, Encryption extensions, Singleton pattern template, and string and list extensions.

CpUnityExtensions - Unity-specific extensions. Some helpful extensions functions for Unity-specific data types.

CpTweening - Custom tweening framework for Unity. Animate anything: color, size, position, particles settings, and much more. Can easily be extended to your own needs.

World is your oyster. Create and have fun!
