Detailed Description

Signal mapping automatically connects injected signals to matching listener member methods. Signal mapping works by extracting the name of a signal property and finding a method that matches the name pattern and parameters of a signal. To be able to automap, signal property must be named with "Signal" suffix, e.g. LevelLoadedSignal. Method must contain same base name, but instead have "On" prefix, e.g. OnLevelLoaded.
