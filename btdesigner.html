Behavior Tree Designer

(in development)

Behavior tree designer (better name coming soon) is a tool for creating behavior trees in Unity3D.

While Unity3D has plenty behavior designers on asset store, they are all very closely integrated with Unity. This makes them great for single player games, however unusable when there is a non-Unity server. Our BTD is built with such use case in mind and is designed to be modular. The interface that creates and debugs behaviors runs in Unity, however its backend logic is not dependent on Unity engine.
