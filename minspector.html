Mosaic Inspector

(in development)

Mosaic Inspector (MI) is a data-driven property inspector and user interface generator for Unity. It uses a set of descriptors and attributes to create an interactive UI from reflected data. MI’s main goal is to facilitate creating game debug tools, map editors, in-game inspectors, and visualizing any necessary data without having to manually build UI or rely on Unity’s Inspector panel.

  • Facilitate creating game debug tools, map editors, in-game inspectors.
  • Visualize any necessary data without having to manually build UI or rely on Unity’s Inspector panel.
  • Design custom UI with included Mosaic Creator tool.
  • Actively maintained. Visit Community for techinical support.
